# Hướng dẫn thêm file âm thanh

Để game hoạt động đầy đủ, bạn cần thêm các file âm thanh sau vào thư mục này:

## Các file cần thiết:

1. **background.mp3** - Nhạc nền vui nhộn cho game
2. **flip.mp3** - Âm thanh khi lật thẻ
3. **match.mp3** - Âm thanh khi ghép đúng cặp thẻ
4. **win.mp3** - Âm thanh khi chiến thắng
5. **lose.mp3** - Âm thanh khi thua/hết giờ

## Nguồn tải file âm thanh miễn phí:

- [Freesound.org](https://freesound.org/)
- [Mixkit.co](https://mixkit.co/free-sound-effects/)
- [ZapSplat.com](https://www.zapsplat.com/)
- [Pixabay Music](https://pixabay.com/music/)

## Gợi ý từ khóa tìm kiếm:

- **background.mp3**: "happy game music", "upbeat background music", "cheerful loop"
- **flip.mp3**: "card flip", "paper flip", "whoosh"
- **match.mp3**: "success", "correct", "ding", "positive"
- **win.mp3**: "victory", "win fanfare", "success jingle"
- **lose.mp3**: "fail", "game over", "negative buzz"

## Lưu ý:

- Định dạng: MP3
- Dung lượng: Nên dưới 500KB cho mỗi file để tải nhanh
- Background music nên có độ dài khoảng 30-60 giây và có thể loop
- Các sound effect nên ngắn (1-2 giây)
